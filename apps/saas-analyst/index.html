Claude Code Skill · Anthropic

SaaS Analyst

深度调研任意 SaaS 产品，自动生成竞品矩阵、定价策略分析与增长路径报告。
一条命令，完整的 SaaS 情报包。

竞品分析 功能矩阵 定价策略 增长路径 用户画像

输出内容

  • ▸
    竞品全景矩阵
    主要竞品的功能、定价、目标客户、市场定位横向对比表
  • ▸
    定价策略解析
    套餐结构、价格锚点、Free/Trial 设计、付费转化逻辑
  • ▸
    增长路径分析
    获客渠道、留存机制、扩张路线图及关键里程碑
  • ▸
    用户与市场画像
    ICP 定义、痛点地图、决策链与购买触发器

适合场景

  • ·进入新市场前的竞品摸底
  • ·产品定价调研与重新定价决策
  • ·投资尽调中的 SaaS 标的快速分析
  • ·GTM 策略制定前的市场情报收集
  • ·产品经理竞品周报自动化

运行环境

  • Claude Code ≥ 1.0
  • Anthropic API Key
  • 网络访问（调研时实时搜索）

使用方法

Step 1
安装 Claude Code
从 claude.ai/download 下载并登录
Step 2
激活 Skill
/anthropic-skills:saas-analyst
Step 3
输入目标产品
告诉 Claude 你要调研哪款 SaaS 产品

示例 Prompt

/anthropic-skills:saas-analyst

调研 Linear（项目管理工具）：
- 重点分析 vs Jira / Notion / Asana 的差异
- 定价结构和 Free tier 设计逻辑
- 近两年的增长策略

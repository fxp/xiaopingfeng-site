Privacy Policy / 隐私政策

Last updated: 2026-04-23 · Hypothesisor v0.1.8

What we collect / 我们收集什么

Nothing. Hypothesisor 本身不收集、不存储、不传输任何用户数据到我们自己的服务器。我们没有服务器。

What stays on your machine / 留在你本机的东西

  • 你在设置页填的 Hypothesis Token 与 BigModel API Key，存于 chrome.storage.local，仅用于扩展自身访问对应 API。永不外传。
  • 你选择的默认 mode / style 偏好，同样存在 chrome.storage.local。

What we send out / 哪些数据会离开你的浏览器

  1. 当前网页正文（通过 chrome.scripting.executeScript 抽取） + 你选择的 mode/style → BigModel API（open.bigmodel.cn），用于生成标注候选。
  2. 你勾选发布的标注（quote + comment + tags + 当前页 URL） → Hypothesis API（api.hypothes.is），用于发布到你的 Hypothesis 账户。

这两个目的地受各自服务的隐私政策约束：

  • 智谱 AI / BigModel 隐私政策
  • Hypothesis Privacy Policy

Third parties / 第三方

除上述两个 API 外，扩展不向任何第三方（包括我们）发送数据，不嵌入跟踪脚本，不使用 cookie。

Permissions justification / 权限说明

  • activeTab + scripting：抓取你主动激活的页面正文，仅在你点扩展按钮时执行。
  • storage：本地存 Token / API Key / 偏好。
  • host_permissions 仅限 open.bigmodel.cn 与 api.hypothes.is，仅用于 API 调用。

Open source / 开源

代码完全开源在 github.com/fxp/Hypothesisor，可自行审计。

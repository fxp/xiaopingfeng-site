## 五个反共识洞察

7 篇文章里贯穿的反共识洞察：

**1. 模型能力本身已不是护城河**——Opus 4.7 vs GPT-5.5 差距压缩到 1.1 个百分点 + 7 天。真正的护城河是协议 + 平台 + 生态 + 政策四层叠加。

**2. SaaSpocalypse 受益最大者不是开源 AI，是 Anthropic**——蒸发的 $285 亿原本属于"thin wrapper"型 SaaS，最终回流的是 Anthropic 的 token 经济。

**3. Anthropic 反集权条款不是道德负担，是商业资产**——付出数亿美元联邦合同代价，换来"独立可信"叙事在金融 / 医疗 / 教育 / 国际市场的几百亿溢价。

**4. 主动披露负面对齐研究是反共识的最佳公关**——Sleeper Agents、Alignment Faking、96% 勒索率这些"听上去糟糕"的论文，实际上把 Anthropic 推上了"行业最严肃公司"的位置。

**5. 30 天无 post-mortem 是被低估的最大品牌损伤**——比 post-mortem 内容本身更具杀伤力。它把"AI 安全"从 Anthropic 的核心叙事拆解为窄定义。

## 30 秒读完：三层栈框架

| 层级 | 内容 | 毛利分布 |
|------|------|---------|
| **底层（算力 / 半导体）** | NVIDIA / TSMC / Broadcom / 数据中心 | 极高毛利，但资本开支也极重 |
| **中层（模型 / 基础设施）** | OpenAI / Anthropic / Google / Meta | 高研发烧钱，毛利受 token 价格战压制 |
| **应用层（垂直 SaaS / Agent）** | Cursor / Glean / 各种 vertical AI | 毛利取决于差异化能力，多数为代销中层 API |

**核心洞察**：毛利沿着栈分布并不均匀——**底层吃最多，应用层吃最少，中层在中间被压**。这个分布解释了为什么 Anthropic / OpenAI 同时进军应用层（Antspace / Workspace Agents）。

---

## 这门课最值得记住的三个框架

| # | 框架 | 含义 |
|---|------|------|
| 1 | **三层栈** | AI 产业不是单一市场，是底层/中层/应用层三个不同经济结构的叠加 |
| 2 | **毛利分布** | 越靠近物理硬件毛利越高；越靠近终端用户毛利越低；中层被两头压 |
| 3 | **瓶颈模型** | 任意时点 AI 行业的瓶颈不是均匀分布——可能是算力（2024-2026）、可能是数据（2026 中起 Karpathy 论）、可能是电力（2026 后） |

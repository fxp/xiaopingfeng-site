// 招募对象

X型AI少年

内驱力，而不是成绩单。能"停不下来"谈AI的孩子，比"门门都A"的孩子更适合这里。

🔥
对某个AI问题停不下来
能滔滔不绝地讲一个你想用AI解决的问题，根本不需要人催你——这是内驱力最真实的体现，也是我们最看重的。
💥
把失败当成数据
API报错、模型幻觉、逻辑跑偏——这些都是学习材料。我们要找那种代码崩了会问"为什么"而不是放弃的人。
🌀
在不确定里越钻越兴奋
AI领域没有标准答案。我们需要那种面对模糊问题会更兴奋、而不是需要老师告诉他"答案是什么"的人。
🛠️
想法立刻想变成东西
你脑子里有个想法，第一反应是"怎么做出来"，而不是"这个想法好不好"。动手能力是这里最核心的货币。

  • 希望老师每天告诉他该做什么的

  • 需要分数和排名驱动才能学习的

// 学习模式

Coach，不是老师

传统老师教你答案。Coach帮你找到值得回答的问题。这是两件完全不同的事。

传统老师做的 我们的 Coach 做的
布置任务，检查完成问"你为什么想解决这个问题"
讲授知识，学生接收当最尖锐的批评者，攻击你的方案
给分数，定好坏问"用户真的需要这个吗"
担心学生走弯路让你走弯路，因为那是最快的路
给出标准答案说"我不知道，你去找出来"
每周讲课2小时每天随时在线，随时打断你
Coach 会做的一件具体的事：当你来说"我想做一个AI音乐推荐器"，Coach 的第一个问题不是"技术怎么实现"，而是——"你发现了什么真实的痛点？谁在真实地遇到这个问题？" 然后陪你把问题想清楚，再放你去做。

// 项目路径

从问题到作品

28天。不是三个学科，不是四个模块。是一件你真正想做的事，从头做到底。

PHASE 01 · DAY 1-7
X-Challenge
找到你的问题
// 一周，找到值得你花28天的事
不急着做东西。先花一整周，搞清楚你真正想解决的是什么。大多数失败的项目，都是因为做的不是真实问题。
  • AI边界测绘：找出AI擅长和不擅长的边界
  • 现实世界观察：找3个让你真正烦恼的场景
  • 用户访谈：和5个真实的人聊，记录痛点
  • 问题陈述：一句话说清楚"谁在什么情况下无法做什么"
  • Coach 签字确认后，才能进入构建阶段
📌 产出：问题陈述 · Coach 确认书
PHASE 02 · DAY 8-21
Build Sprint
构建 · 测试 · 迭代
// 两周，用无限弹药做出真实的东西

// 常见问题

你可能想问的

还有问题，直接发邮件给我们。

我不会写代码，能来吗？ ▾
可以。我们有不少学员进来时几乎不会写代码。有了 Cursor + Claude，代码是可以被学会的——关键是你有没有一个真实的问题驱动你去学。不会写代码但有强烈问题意识的人，比会写代码但不知道做什么的人更适合这里。
"无限量Token"是认真的吗？ ▾
认真的。我们不希望你在做决策时有任何关于"省着用"的顾虑。API费用由我们统一承担。唯一的前提是：你真的在做你自己的项目，不是拿来做无关的事情。到目前为止，没有任何一个认真做项目的学员达到过让我们担心的额度。
和普通AI夏令营有什么不同？ ▾
普通AI夏令营：老师给任务，学生完成，结营有证书。

Project School：没有固定任务，你找自己的问题，用我们给的弹药做出来，让真实的用户去用。我们不给作业，我们给车库和弹药。
如果28天我没做完怎么办？ ▾
"没做完"是正常的。重要的是：你有没有每天都在认真推进。28天结束时，我们看的是你的决策质量和学习过程，不是你的产品功能数量。做了一半但学到了很多，远比"做完了但什么都没想清楚"更有价值。
线上还是线下？ ▾
目前主要以线上形式进行（Discord + 每日视频站会）。部分期次会在北京、深圳设置线下共学空间，优先向当地学员开放。最终的 Demo Day 会有线下公开活动。
费用是多少？ ▾
Project School 目前处于早期阶段，对第一批学员实行大幅优惠学费，所有 API 费用和工具订阅由我们承担。具体学费在申请后根据情况沟通。我们会确保费用不是有能力的孩子进来的障碍。

你已经有一个停不下来的想法了吗？

那就是你应该来的理由。不需要简历，不需要成绩单。告诉我们那个让你睡不着觉的AI问题。
